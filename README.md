### important keys:
#### IUser.User  
- login 用户名  
- html_url 主页url
- avatar_url 头像url  

### IRepository.Repo  
- name 仓库名  
- owner 所有者信息
- private 是否私有  
- description  描述
- html_url 主页url  
- clone_url `git clone`的地址  

### IBranch.BranchDetail
- commit.commit.author 作者信息  
- commit.commit.committer 提交者信息  
- commit.commit.tree 获取文件树(一层)
